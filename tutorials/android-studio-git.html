Configuring Android Studio for a git project

Prerequisite

I am assuming that you already have Android Studio and git installed on your computer. If you are on Windows, installing Babun allows you to run command with the Cygwin environment, and also installs git by default (see How to install Babun).

Windows only: configure Android Studio to find git

On the "Welcome to Android Studio" window, go to Configure > Preferences > Version Control > Git.

Set the "Path to Git executable" to the location of git (which you can find using the which git command. If you are on Windows, it is probably C:\Users\_YOUR_NAME_\.babun\bin\git).

Importing a GitHub Android Studio project

Go back to the "Welcome to Android Studio" window, and click "Check out from Version Control", then GitHub.

Type the host (github.com or github.company.com if you are using GitHub enterprise) and your credentials.

If you want Android Studio to remember your password with: "Save password", you can set a master password (you will need to type this in order for Android Studio to access your previous password), or just leave it blank.

Choose the GitHub repository you want to clone (copy) to your computer

Choose where to copy it (the parent directory must not have the same directory already in it).

You have now successfully configured Android Studio to link your Android project to GitHub.
