Getting ready for ROR on Windows

Install Babun

Go to http://babun.github.io/ and follow the instructions to install Babun.

Configure babun

Run the following commands in the Babun shell:

babun update
# Next line is one big line
curl -sSL https://raw.githubusercontent.com/xjlin0/getrbenv-installer/feature/fix_zshrc_install_dir_pathfix/bin/install.sh | bash -s -- --plugins sstephenson/ruby-build,rkh/rbenv-update
pact install sqlite3

Close and restart Babun shell.

Install Ruby and ROR

Run the following commands to install Ruby and ROR:

rbenv install 2.3.0
rbenv rehash
rbenv global 2.3.0
gem install rails

Now try to run the following commands, and make sure they work:

sqlite3 --version
rails --version

In case of problems, have a look at this more complete tutorial: https://xjlin0.github.io/tech/2015/04/14/babun-the-new-cygwin-for-ruby-rails-sinatra-and-nodejs/.
