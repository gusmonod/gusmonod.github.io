Installing and configuring Babun

Install Babun

Go to http://babun.github.io/ and follow the instructions to download Babun.

Unzip Babun and launch the install.bat file. This should open a cmd window and print a lot of text.

If you have an error that says something like .babun folder already exists, then you can delete this folder (in C:\Users\_YOUR_NAME_) and everything it contains.

Configure Babun

Run the following commands in the Babun shell:

babun update

Use Babun

You are now ready to use Babun! Babun comes with a lot of good preinstalled packages, including git, zsh with oh-my-zsh, vim, and a lot of others. But IMO, the best feature of Babun over Cygwin is its ability to install and update packages directly from the command line.

pact install _YOUR_DREAM_PACKAGE_HERE_

Have fun using Babun (sorry you're on Windows).
